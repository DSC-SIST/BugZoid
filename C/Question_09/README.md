# Question 9

## Problem Statement

A program to 

- sort the elements at odd positions in descending order
- sort the elements at even positons in ascending order.

## Difficulty Level 

<b>Hard</b>

## Sample Testcases

#### Example 1

##### Input

13,2,4,15,12,10,5

##### Output

13,2,12,10,5,15,4

#### Example 2

##### Input

1,2,3,4,5,6,7,8,9

##### Output

9,2,7,4,5,6,3,8,1
