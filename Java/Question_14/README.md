## **Difficulty Level**

&nbsp;&nbsp; EASY

## **Bug Type**

- Logical Errors

## **Sample Input**

&nbsp;&nbsp; Enter first number: 24<br>
&nbsp;&nbsp; Enter second number: 12

## **Sample Output**

&nbsp;&nbsp; Quotient: 2<br>
&nbsp;&nbsp; Remainder: 0
